# ArticleWorld - 文章撰寫生態系統

## 快速啟動

使用本技能將自動啟動 **article-consult**（總協調者），他會：

1. 釐清您的內容需求
2. 調度團隊完成製作流程
3. 確保品質達到 8/10 分以上
4. 交付完整的內容素材包

## 使用方式

```
/ArticleWorld 寫一篇關於 [主題] 的文章
```

或直接提供素材：

```
/ArticleWorld

我有以下素材：
[貼上對話記錄/會議紀錄/隨筆]

請幫我整理成文章
```

## 核心特色

### 1. 七大寫作風格系統

可選擇以下風格：
- **教練風格** - 提問引導、自我覺察
- **故事風格** - 細節沉浸、情感共鳴
- **商業風格** - 數據驅動、框架清晰
- **勵志風格** - 情感賦能、信念改變
- **厚黑學風格** - 揭露人性、冷靜分析
- **哲學反思風格** - 深度思考、拆解重建
- **Blog 風格** - 個人觀點、靈活表達

### 2. 去 AI 味機制

自動檢測並避免：
- 空洞轉折詞（「不是...而是...」）
- 罐頭結尾（「總的來說」）
- 宏大修飾（「多維度」「賦能」）

確保文章有：
- 具體細節（時間、地點、數字）
- 作者聲音（立場、情緒）
- 不完美感（承認不確定）

### 3. 品質門檻把關

所有內容必須達到 **8/10 分**以上：
- 受眾吸引力評分
- 非受眾擴散力評分
- 平台適配度評分（FB/IG/Blog/LinkedIn）

未達標自動退回修正。

### 4. 內容誠信保證

AI 絕對不會編造：
- 場景、故事
- 對話、經驗
- 時間、地點、人物

所有具體細節來自您提供的素材。

## 適用內容類型

- 長文章（深度分析、專題報導）
- 部落格文章（經驗分享、觀點評論）
- 行銷文案（產品介紹、品牌故事）
- 社群短文（FB/IG/LinkedIn 貼文）
- EDM 文案（電子報內容）
