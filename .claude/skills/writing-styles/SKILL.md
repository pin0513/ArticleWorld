# 七大寫作風格系統

## 快速參考

本技能提供七種專業寫作風格的完整指南，每種風格都包含：
- 核心 DNA 與特徵
- 英文世界與華文世界代表作者
- 實際應用範例
- 風格切換技巧

## 七大風格速查

| 風格 | 核心問句 | 說服路徑 | 讀者帶走什麼 |
|------|---------|---------|------------|
| **教練** | 「你自己怎麼看？」 | 提問→自我覺察→行動 | 一個新的自我認識 |
| **故事** | 「然後呢？」 | 細節→沉浸→頓悟 | 一個忘不掉的畫面 |
| **商業** | 「怎麼做才有效？」 | 數據→認知改變→行動 | 一個可執行的框架 |
| **勵志** | 「怎麼活才有意義？」 | 情感→信念改變→行動 | 一份勇氣或力量 |
| **厚黑學** | 「人性的真相是什麼？」 | 揭露→警醒→防禦/運用 | 一副看穿人心的眼鏡 |
| **哲學反思** | 「這一切意味著什麼？」 | 懷疑→拆解→重建 | 一種更清醒的活法 |
| **Blog** | 「我怎麼看這件事？」 | 觀點→展開→印記 | 一個新觀點或操作手冊 |

---

## 使用方式

### 選擇風格

根據內容目的選擇適合的風格：

**需要讀者自我覺察** → 教練風格
**需要情感沉浸** → 故事風格
**需要提供方法論** → 商業風格
**需要情感共鳴與賦能** → 勵志風格
**需要揭露人性真相** → 厚黑學風格
**需要深度思考** → 哲學反思風格
**需要個人觀點表達** → Blog 風格

### 混合使用

頂尖作者常跨界使用多種風格：
- 商業 + 故事：用案例支撐框架
- 教練 + 勵志：提問引導 + 情感賦能
- 哲學反思 + 故事：深度思考 + 具體場景

## 風格切換技巧

### 開場句範例

| 風格 | 典型開場 |
|------|---------|
| 教練 | 「你有沒有想過......？」 |
| 故事 | 「2023年3月15日下午三點半......」 |
| 商業 | 「哈佛商學院研究了312家企業，發現......」 |
| 勵志 | 「我曾經也以為......直到那一天......」 |
| 厚黑學 | 「世人皆以為......其實......」 |
| 哲學反思 | 「我們以為確定的，其實......」 |
| Blog | 「這幾天在想一個問題......」 |

### 結尾句範例

| 風格 | 典型結尾 |
|------|---------|
| 教練 | 「所以，你的答案是什麼？」 |
| 故事 | （戛然而止，留下畫面） |
| 商業 | 「三個行動步驟：1. ... 2. ... 3. ...」 |
| 勵志 | 「你也可以。現在就開始。」 |
| 厚黑學 | （冷酷的結論，不留情面） |
| 哲學反思 | 「或許，這才是問題的開始。」 |
| Blog | 「就這樣。」或提出一個問題 |

## 品質檢查清單

寫完後，確認：

✅ 風格是否一致？（不要中途換風格）
✅ 是否符合該風格的核心 DNA？
✅ 語氣是否到位？
✅ 讀者能否帶走該風格承諾的東西？
