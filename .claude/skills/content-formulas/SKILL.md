# 內容創作公式集

## 文章結構公式

### 1. 問題驅動型（Problem-Solution）

```
開場：提出讀者痛點
   ↓
加深：描述問題的嚴重性
   ↓
轉折：介紹解決方案
   ↓
說明：如何實施
   ↓
收尾：行動呼籲
```

**適用**：教學文、商業文、工具介紹

---

### 2. 故事驅動型（Storytelling）

```
鉤子：吸引人的開場（場景/對話/反差）
   ↓
建立：背景與人物
   ↓
衝突：遭遇問題/挑戰
   ↓
轉折：關鍵決定/頓悟
   ↓
結局：結果與啟示
```

**適用**：個人經驗、案例分析、品牌故事

---

### 3. 框架驅動型（Framework-Led）

```
宣言：核心觀點（反直覺更好）
   ↓
框架：3-5個概念/步驟
   ↓
逐一展開：每個配案例
   ↓
整合：如何搭配使用
   ↓
總結：可執行的下一步
```

**適用**：商業文、方法論、思維模型

---

### 4. 對比驅動型（Contrast）

```
常見做法：大多數人的方式
   ↓
為什麼不行：拆解問題
   ↓
更好的方法：提出替代方案
   ↓
證據：數據/案例/原理
   ↓
行動：讀者如何應用
```

**適用**：顛覆認知、挑戰傳統、教育讀者

## 文案公式

### AIDA（經典銷售公式）

| 階段 | 目的 | 技巧 |
|------|------|------|
| **A**ttention | 吸引注意 | 強烈標題、反差、問句 |
| **I**nterest | 引起興趣 | 關聯痛點、數據、故事 |
| **D**esire | 激發渴望 | 好處、對比、證據 |
| **A**ction | 促成行動 | CTA、緊迫感、簡化 |

**範例**：
```
標題：「為什麼90%的待辦清單都沒用？」（Attention）
正文：「你是否也有寫不完的清單...」（Interest）
    「試試這個方法，一天只做三件事...」（Desire）
結尾：「現在就列出明天的三件事」（Action）
```

---

### PAS（痛點放大公式）

| 階段 | 說明 |
|------|------|
| **P**roblem | 點出問題 |
| **A**gitate | 放大痛苦 |
| **S**olution | 給出解法 |

**範例**：
```
P: 「你寫的文章沒人看」
A: 「花了三小時寫，按讚只有三個。你的努力被演算法埋沒。」
S: 「用這個標題公式，點擊率提升300%」
```

---

### BAB（前後對比公式）

| 階段 | 說明 |
|------|------|
| **B**efore | 之前的困境 |
| **A**fter | 之後的美好 |
| **B**ridge | 如何跨越（方法） |

**範例**：
```
Before: 「以前寫文章要三小時」
After:  「現在用框架，30分鐘搞定」
Bridge: 「只要掌握這三個結構...」
```

---

### 4P（說服公式）

| 階段 | 說明 |
|------|------|
| **P**icture | 描繪畫面 |
| **P**romise | 給予承諾 |
| **P**rove | 提供證據 |
| **P**ush | 推動行動 |

## 開場鉤子公式

### 1. 反直覺陳述

```
「你以為 [常識]，其實 [反轉]」
範例：「你以為多工很有效率，其實讓你變笨了」
```

### 2. 震撼數據

```
「[驚人數字] 的人 [令人意外的事實]」
範例：「87% 的創業者在第一年就放棄了夢想」
```

### 3. 場景重建

```
「[具體時間]，[具體地點]，[具體人物] [在做什麼]」
範例：「2023年3月15日下午三點，星巴克二樓靠窗的位置，我刪掉了工作五年的專案」
```

### 4. 直接問句

```
「你是否也 [痛點]？」
範例：「你是否也覺得，每天都很忙，但沒做什麼有意義的事？」
```

### 5. 個人告白

```
「我曾經以為 [錯誤認知]，直到 [轉折事件]」
範例：「我曾經以為努力就會成功，直到連續被拒絕50次」
```

## 結尾公式

### 1. 行動呼籲

```
「現在就 [具體行動]」
範例：「現在就打開筆記本，寫下三個目標」
```

### 2. 開放式問句

```
「那麼，你的答案是什麼？」
範例：「所以，你願意為了夢想冒險嗎？」
```

### 3. 戛然而止

```
（不下結論，留下畫面或情緒）
範例：「他關上電腦，走進雨中。」
```

### 4. 回扣開頭

```
（回到開場的場景/問題，但帶著新的理解）
範例：「還記得文章開頭的那個問題嗎？現在你知道答案了。」
```

## 標題公式

### 數字型

```
「[數字] 個 [方法/秘密/錯誤] [達成目標]」
範例：「7個讓文章被瘋傳的標題技巧」
```

### 問句型

```
「為什麼 [常見現象]？」
範例：「為什麼聰明人也會做蠢事？」
```

### 對比型

```
「不要 [錯誤做法]，試試 [正確做法]」
範例：「不要用待辦清單，試試時間方塊法」
```

### 秘密型

```
「[權威/成功者] 不會告訴你的 [秘密]」
範例：「頂尖文案不會告訴你的寫作公式」
```

### 痛點型

```
「如果你 [痛點]，這篇文章可以幫你」
範例：「如果你寫文章總是卡住，這篇文章可以幫你」
```

## 使用指南

### 選擇公式

| 內容類型 | 推薦公式 |
|----------|---------|
| 教學文 | 問題驅動型 + AIDA |
| 個人經驗 | 故事驅動型 + BAB |
| 商業文 | 框架驅動型 + 4P |
| 行銷文案 | PAS + 數字型標題 |
| 觀點文 | 對比驅動型 + 反直覺開場 |

### 注意事項

⚠️ **不要生硬套用**：公式是骨架，需要血肉（故事、細節、情感）
⚠️ **不要過度承諾**：標題吸睛，但內容要能兌現
⚠️ **不要全用公式**：頂尖作者會打破公式，創造驚喜
