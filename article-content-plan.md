## 二、Skills 清單與職責

### 主協調者

| Skill | 職責 | 類型 |
|-------|------|------|
| `article-consult` | 全流程協調、輸入主題/素材、輸出企劃案 | 新建 |

### Phase 1：企劃與研究

| Skill | 職責 | 類型 |
|-------|------|------|
| `article-planner` | 內容企劃、主題定調、受眾分析、最終企劃案輸出 | 新建 |
| `article-researcher` | 主題研究 (Wiki/學術/社會風向/正反面/產業趨勢) | 新建 |

### Phase 2：撰文與編輯

| Skill | 職責 | 類型 |
|-------|------|------|
| `article-writer` | Storytelling、哲學反思、學習教練視角、產業想像 | 新建 |
| `article-editor` | 總編輯審稿、潤稿、標題腦力激盪 | 新建 |

### Phase 3：行銷與分析

| Skill | 職責 | 類型 |
|-------|------|------|
| `article-marketing` | 標題優化、行銷文案、關鍵字/Hashtag 建議 | 新建 |
| `article-analytics` | 熱門貼文分析、受眾評分、品質門檻把關 | 新建 |

### 可複用現有 Skills

| Skill | 複用方式 |
|-------|----------|
| `/slide-image` | 文章配圖生成 |
| `/product-strategy-coach` | 產業觀點審視 |

### 雙向串連：article-researcher ↔ slide-research

```
┌─────────────────────┐         ┌─────────────────────┐
│  article-researcher │ ←─────→ │   slide-research    │
│    (文章研究員)      │         │    (簡報研究員)      │
└─────────────────────┘         └─────────────────────┘
         │                               │
         ▼                               ▼
┌─────────────────────┐         ┌─────────────────────┐
│ 專注：               │         │ 專注：               │
│ • 社會風向/輿論      │         │ • 數據佐證/統計      │
│ • 正反論述/爭議面    │         │ • 引用來源/權威性    │
│ • 哲學/反思角度      │         │ • 案例研究/成功故事  │
│ • 有趣/冷知識        │         │ • 視覺化資料建議     │
│ • 學習啟發點         │         │ • 產業報告/市調      │
└─────────────────────┘         └─────────────────────┘
```

**分工原則**：
- `article-researcher`：偏向「觀點、反思、社會性」研究
- `slide-research`：偏向「數據、佐證、權威性」研究
- 兩者可互相呼叫，但各自輸出不重複
- 共享研究成果時，標註來源技能

## 三、詳細流程設計

### Phase 1：企劃與研究

```
[輸入]                    [處理]                      [輸出]
───────────────────────────────────────────────────────────────
主題/素材               article-planner            內容企劃書
├─ 一段對話              ├─ 主題定調                ├─ 文章定位
├─ 文章素材              ├─ 受眾分析                ├─ 目標受眾
├─ 靈感關鍵字            ├─ 角度選擇                ├─ 內容大綱
└─ 產業主題              └─ 風格決定                └─ 研究需求
                              │
                              ▼
                        article-researcher
                         ├─ Wiki 定義
                         ├─ 學術觀點
                         ├─ 社會風向
                         ├─ 正反論述
                         ├─ 有趣面向
                         ├─ 社會/環境/人類影響
                         ├─ (可選) 政治趣味提示
                         └─ (可選) 產業中長期想像
```

### Phase 2：撰文與編輯

```
[輸入]                    [處理]                      [輸出]
───────────────────────────────────────────────────────────────
內容企劃書              article-writer              初稿
研究資料                 ├─ Storytelling
                         ├─ 哲學反思 (視主題)
                         ├─ 學習教練視角 (視主題)
                         │   ├─ 個人學習
                         │   ├─ 團隊學習
                         │   └─ 組織學習
                         └─ 產業想像 (視主題)
                             ├─ 3年內趨勢
                             └─ 5年後願景
                              │
                              ▼
                        article-editor
                         ├─ 內容審查
                         ├─ 文字潤稿
                         ├─ 標題腦力激盪 (與 writer)
                         └─ 輸出：審稿後文章 + 標題候選
                              │
                              ▼
                        article-planner (回審)
                         ├─ 主題明確性確認
                         ├─ 受眾適配確認
                         └─ 輸出：定稿文章
```

### Phase 3：行銷與分析

```
[輸入]                    [處理]                      [輸出]
───────────────────────────────────────────────────────────────
定稿文章                article-marketing           行銷素材包
標題候選                 ├─ 標題優化/修正
                         ├─ 長標題 (SEO)
                         ├─ 短標題 (社群)
                         ├─ FB 文案 + Hashtag
                         ├─ IG 文案 + Hashtag
                         ├─ (可選) FB/IG 串接文案
                         ├─ 文章正文 (含 Meta)
                         └─ 關鍵字建議
                              │
                              ▼
                        article-analytics
                         ├─ 熱門貼文數據分析
                         ├─ 受眾評分 (1-10)
                         ├─ 非受眾評分 (1-10)
                         ├─ 平台適配度 (FB/IG/Threads/X)
                         └─ 改善建議
                              │
                              ▼
                        ┌─────────────────┐
                        │ 分數 ≥ 8/10 ?   │
                        └────────┬────────┘
                           Yes   │   No
                            ▼    │    ▼
                        [發布]   │  [退回各環節]
                                 └─→ article-planner (重新企劃)
                                 └─→ article-writer (重新撰寫)
                                 └─→ article-editor (重新潤稿)
                                 └─→ article-marketing (重新行銷)
```
